## MVP 一句话定义

一个帮助用户把“探店体验”以**结构化复盘**方式记录下来，并支持**公开发布/收藏/搜索**的移动端应用；同时内置**易发现的产品反馈渠道**与**最小内容治理闭环（举报）**。

## 版本约定

- **MVP**：能稳定记录/检索/公开发布/收藏/举报/反馈；不做复杂社交与评论体系
- **V1**：增加小组/圈子协作（可选）、共同维护店铺、轻量版本记录
